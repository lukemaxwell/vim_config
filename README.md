# vim_config

vimrc used for dev.

## Install Vundle
```
$ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

## Install pluins
```
$ vim +PluginInstall +qall
```

## Disable syntastic for single file
```
:SyntasticToggleMode
```

## Enable syntastic for single file
```
:SyntasticCheck
```
